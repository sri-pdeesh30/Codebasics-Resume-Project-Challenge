# Limitations

- Vote-share erosion does not directly confirm voter transfer between parties.

- ECI data explains electoral outcomes but not voter motivations.

- No demographic, caste, religion, or socioeconomic assumptions were made.

- Regional classifications are analytical groupings and may simplify constituency-level variation.

- Alliance-level political effects were not separately modeled.

- This analysis uses only publicly available ECI election data as required by the challenge guidelines.
